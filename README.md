**IndoTrace-OSINT** is a professional-grade command-line tool designed to trace and gather intelligence on Indian mobile phone numbers. It extracts real-time data including Network Operator, Telecom Circle (State), Signal Status, and even generates Google Maps coordinates based on the registration data.

## ✨ Key Features
- 🚀 **Country Specific**: Optimized strictly for the Indian (+91) numbering plan.
- 📡 **Live Data**: Fetches data directly from telecom databases.
- 📍 **Map Integration**: Automatically generates Google Maps links for the registration area.
- 🚫 **Nokos Detection**: Intelligent logic to identify inactive or unallocated numbers.
- 🎨 **Colorized Interface**: Clean and professional terminal UI with color-coded results.
